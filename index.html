Indikator big data sebagai proksi PDRB kewilayahan

Tiga dashboard ini menyajikan hasil eksplorasi penggunaan data satelit dan data lalu lintas kapal—sumber data gratis dan terbuka—sebagai pelengkap data resmi Produk Domestik Regional Bruto (PDRB) Kabupaten Rembang, terutama untuk gambaran tingkat kecamatan yang belum memiliki angka PDRB resmi.

Periode representatif: 2024–2026

Tentang proyek ini

PDRB resmi hanya tersedia di tingkat kabupaten/kota dan membutuhkan survei yang mahal serta memakan waktu. Eksplorasi ini mencoba memanfaatkan jejak digital—citra satelit dan data lalu lintas kapal—sebagai pelengkap, bukan pengganti, data resmi tersebut.

Setiap dashboard menampilkan data historis, peta perbandingan antar kecamatan, dan (bila tersedia) hasil korelasi dengan data PDRB resmi Kabupaten Rembang—termasuk korelasi yang lemah, karena hal tersebut juga merupakan temuan yang penting untuk dipahami.

Data dapat diperbarui secara mandiri menggunakan skrip Python di folder scripts/. Lihat README untuk panduan lengkap.
